<h1>Sample Backend Implementation for an employee management system</h1>

<h3>After Cloning the repository do the following things</h3>
<ol>
	<li>do "dotnet restore" to download all dependencies</li>
	<li>start application</li>
	<li>Use localhost:5190 for the backend</li>
	<li>Use https://localhost:7051/swagger for swagger api (testing api through browser UI)</li>
</ol>
